**1. Create `src/stores/configStore.js`** -- Zustand store replacing the Config class singleton.

Convert the Config class to a Zustand store with these specifics:
- Use `import { create } from 'zustand'` and `import { persist } from 'zustand/middleware'`
- localStorage key: `"charGeneratorConfig"` (same as original `LOCAL_STORAGE_KEY`)
- Session storage keys (same as original `SESSION_STORAGE_KEYS`):
  - `"charGeneratorConfig:textApiKey"`
  - `"charGeneratorConfig:imageApiKey"`
  - `"charGeneratorConfig:stPassword"`
- Default config shape MUST match original `getDefaultConfig()` exactly:
  ```javascript
  {
    api: {
      text: { baseUrl: '', apiKey: '', model: '', visionModel: '', timeout: 180000 },
      image: { baseUrl: '', apiKey: '', model: '', size: '', timeout: 180000 },
      sillytavern: { url: '', password: '' },
    },
    app: { maxRetries: 3, retryDelay: 1000, debugMode: false, persistApiKeys: false, enableImageGeneration: true },
    prompts: { contentPolicyPrefix: false, overrides: {} },
  }
  ```
- Use Zustand `persist` middleware for localStorage. API keys and passwords MUST be excluded from localStorage persistence using the `partialize` option -- strip `api.text.apiKey`, `api.image.apiKey`, `api.sillytavern.password` before persisting.
- Implement sensitive value storage separately: `persistSensitiveValues()` and `restoreSensitiveValues()` methods that read/write session or localStorage based on `app.persistApiKeys` flag (matching the original Config class behavior exactly).
- Provide a `get(path)` method for dot-notation access (e.g., `get('api.text.baseUrl')`) to ease migration of consumers.
- Provide a `set(path, value)` method that updates nested state and triggers persist.
- Provide `validateConfig()` returning string[] of errors (same logic as original).
- Call `restoreSensitiveValues()` during store initialization (after persist middleware hydrates from localStorage).
- Export both `useConfigStore` (hook for React components) and a `configStore` getter for non-React service modules: `export const configStore = { get: (path) => useConfigStore.getState().get(path), getState: () => useConfigStore.getState() }`.
