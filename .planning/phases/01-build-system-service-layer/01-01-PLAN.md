   ```
   npm install -D concurrently@9
   ```

   ```
   npm uninstall http-server
   ```

5. Update package.json scripts to (per D-06):
   ```json
   {
     "scripts": {
       "dev": "concurrently \"npm run dev:proxy\" \"npm run dev:frontend\"",
       "dev:proxy": "cd proxy && npx nodemon server.js",
       "dev:frontend": "vite",
       "build": "vite build",
       "preview": "vite preview",
       "start": "cd proxy && node server.js"
     }
   }
   ```

6. Update engines field to require Node 20.19+ (per BUILD-06, Vite 8 requirement):
   ```json
   { "engines": { "node": ">=20.19.0" } }
   ```

7. Create `vite.config.js` at project root (per D-04):
   ```javascript
   import { defineConfig } from 'vite';
   import react from '@vitejs/plugin-react';

   export default defineConfig({
     plugins: [react()],
     server: {
       port: 2427,
       proxy: {
         '/api': {
           target: 'http://localhost:2426',
           changeOrigin: true,
         },
         '/health': {
           target: 'http://localhost:2426',
           changeOrigin: true,
         },
       },
     },
     build: {
       outDir: 'dist',
       emptyOutDir: true,
     },
   });
   ```

8. Add `dist/` to `.gitignore` if not already present. Also add `*.local` (Vite env files).
  </action>
